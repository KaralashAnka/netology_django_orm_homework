# Домашнее задание: Работа с ORM в Django

Этот проект содержит решения двух задач по работе с ORM в Django:

1. **Задание 1 (обязательное)**: Выгрузка каталога товаров из CSV-файла
2. **Задание 2 (дополнительное)**: Онлайн-библиотека

## Структура проекта

```
.
├── task1_phones/          # Задание 1: Каталог телефонов
│   ├── manage.py
│   ├── phones/           # Django приложение
│   └── import_phones.py  # Скрипт импорта из CSV
│
└── task2_books/          # Задание 2: Онлайн-библиотека
    ├── manage.py
    └── books/            # Django приложение
```

## Задание 1: Импорт каталога телефонов из CSV

### Описание
Необходимо создать Django-приложение для импорта и отображения каталога телефонов из CSV-файла.

### Функционал
- Импорт данных о телефонах из CSV файла
- Создание slug на основе названия телефона
- Отображение списка всех телефонов
- Детальная страница каждого телефона

### Установка и запуск

```bash
cd task1_phones

# Создание виртуального окружения (опционально)
python3 -m venv venv
source venv/bin/activate  # для Linux/Mac
# или
venv\Scripts\activate  # для Windows

# Установка зависимостей
pip install django

# Применение миграций
python manage.py migrate

# Импорт данных из CSV
python manage.py shell < import_phones.py

# Запуск сервера
python manage.py runserver
```

### Доступные URL
- `http://127.0.0.1:8000/` - список всех телефонов
- `http://127.0.0.1:8000/catalog/<slug>/` - детальная страница телефона

## Задание 2: Онлайн-библиотека

### Описание
Веб-приложение для управления библиотекой книг с датами публикации.

### Функционал
- Список всех книг
- Сортировка по названию и дате публикации
- Детальная страница книги с информацией об авторе

### Установка и запуск

```bash
cd task2_books

# Создание виртуального окружения (опционально)
python3 -m venv venv
source venv/bin/activate

# Установка зависимостей
pip install django

# Применение миграций
python manage.py migrate

# Создание суперпользователя (для добавления книг через админку)
python manage.py createsuperuser

# Запуск сервера
python manage.py runserver
```

### Доступные URL
- `http://127.0.0.1:8000/` - список всех книг (по умолчанию)
- `http://127.0.0.1:8000/?order=name` - сортировка по названию
- `http://127.0.0.1:8000/?order=pub_date` - сортировка по дате публикации
- `http://127.0.0.1:8000/books/<pub_date>/` - детальная страница книги
- `http://127.0.0.1:8000/admin/` - административная панель
